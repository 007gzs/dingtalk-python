### 安装依赖包

```json
certifi==2017.11.5
chardet==3.0.4
idna==2.6
pycrypto==2.6.1
python3-memcached==1.51
requests==2.18.4
six==1.11.0
urllib3==1.22
```

在python的环境中执行：

```shell
pip install -r requirements.txt
```
